# FreeDi-Addon-Modules

Collection of Klipper add-on modules designed to work alongside FreeDi machine configurations.

## Compatibility Requirements

These modules are only supported on printers flashed with FreeDi Firmware and configured with FreeDi base machine configuration files.

They are not intended for non-FreeDi firmware stacks or non-FreeDi base configuration setups.

## Repository Overview

This repository contains reusable configuration modules that extend printer behavior with optional quality-of-life features and effects.

Current modules include:

- Adaptive Pressure Advance
- Bed Soak
- Caselight Effects
